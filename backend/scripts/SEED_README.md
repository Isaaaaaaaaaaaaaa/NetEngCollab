# Seed 数据说明

## 更新内容

本次更新使seed.py生成的测试数据符合增强后的系统功能。

### 新增字段

#### TeacherPost（教师项目）
- **detailed_info**: 项目详细信息（最多10000字符）
  - 包含项目背景、技术要求、工作内容、预期成果等详细描述
  - 每个项目都有丰富的详细信息，便于学生了解项目全貌
  
- **project_status**: 项目状态
  - `recruiting`: 招募中（默认状态）
  - `in_progress`: 进行中（当确认学生数达到recruit_count时自动设置）

#### CooperationRequest（合作请求）
- **student_role**: 学生在项目中的角色（最多64字符）
  - 示例：前端开发、后端开发、算法研究、数据分析、测试、文档撰写、UI设计、项目管理
  - 仅对已确认（confirmed）的合作请求设置
  
- **custom_status**: 自定义状态标签（最多64字符）
  - 示例：进展顺利、需要帮助、已完成阶段任务、待分配任务、学习中
  - 仅对已确认（confirmed）的合作请求设置
